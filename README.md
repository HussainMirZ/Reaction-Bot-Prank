# Human Benchmark Reaction Bot 
A Python script to simulate human-like reaction speeds (120ms - 150ms) to prank your friends on reaction time websites!

### How to use:
1. Install requirements: `pip install pyautogui keyboard`
2. Run the script: `python reaction_speed.py`
3. Hover over the target and press `\` to arm the bot.

*Make sure to choose a button you have on your keyboard*
